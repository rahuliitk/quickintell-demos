# QuickIntell static site

This folder contains a small static website for `quickintell.com` that lists the company's products.

Preview locally:

```bash
# from repository root
cd site
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Deployment:

- Deploy `site/` contents to any static host (GitHub Pages, Netlify, Vercel).
